紹介文

一度だけ青いブロックを打ち出してすべての赤いブロックを落とすゲームです。
ピンクのブロックのステージはどれか一つだけ落とせばクリアです。
全50ステージです。
BGM無し、効果音無しです。
----
Hot Soup Processor 3.7 hgimg4を利用して作成しました。
HSPプログラムコンテスト2016投稿作品のWebGL移植版です。


操作説明

タイトル画面でステージ選択します。
AWDSキーでステージを選択してZキーで開始します。
クリア済みステージには*がつきます。

ステージ画面では
AWDSキーで打ち出し角度を変更します。
Aキー: 半時計回り。Dキー: 時計回り。Wキー: 上を向く。Sキー: 下を向く。
Zキーで青いブロックを打ち出します。
Xキーで同じステージを最初からやり直し。
Cキーでギブアップしてタイトル画面に戻ります。
画面上部をドラッグすると視線を変更できます。

AWDSキーの代わりに←↑→↓キーも使用できます。
Zキーの代わりにスペースキーも使用できます。

パッドにも対応しています。
ボタン1がZキー、ボタン2がXキー、ボタン3がCキーに対応します。
PAD: 0 表示の場合はパッドのボタンを押すと
使えるようになる場合があります。
